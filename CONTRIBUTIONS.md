# Contributing to Keyboard Typing Game

Thank you for considering contributing to this project! 🎉  
We welcome all improvements, bug fixes, and feature suggestions.

---

## How to Contribute

### 1. Fork the repository

Click the **Fork** button on GitHub to create your own copy of the project.

---

### 2. Clone your fork

```bash
git clone https://github.com/your-username/keyboard-game.git
cd keyboard-game
```

### 3. Create a new branch

Always create a separate branch for your feature or fix:

```bash
git checkout -b feature-name
```

Examples:

- `fix-word-bug`
- `add-timer-feature`
- `improve-ui`

### 4. Make your changes

- Keep code clean and readable
- Follow existing project structure
- Test your changes before committing

### 5. Commit your changes

```bash
git add .
git commit -m "Add: short description of changes"
```

Examples:

- `feat: Add timer functionality`
- `fix: Fix word update bug`
- `ui: Improve UI styling`

### 6. Push to your fork

```bash
git push origin feature-name
```

### 7. Create a Pull Request

Go to the original repository and:

- Click “New Pull Request”
- Select your branch
- Describe your changes clearly

## Guidelines

- Keep changes small and focused
- Do not break existing features
- Write clean and readable code
- Be respectful in discussions

## Ideas for Contributions

If you're not sure what to work on, here are some ideas:

- Add typing speed (WPM) calculator
- Add countdown timer mode
- Improve UI/UX design
- Add sound effects for typing
- Add difficulty levels
- Fix bugs or improve performance

## Reporting Issues

If you find a bug:

- Open an issue
- Describe the problem clearly
- Include steps to reproduce it

## Thank You

Your contributions help improve this project and make it better for everyone!
